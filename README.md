# Restaurant Management System üè™

This is a Restaurant Management System using MVC Architecture.

Main Functions are,
 - Place Order
 - Check Out Order
 - Manage Meal
 - Manage Employees
 - Manage Tables
 - Manage Users
 - View Order Details
 - View Reports
 
 And also it has validated input fields and jasper reports and encrypted passwords.
 
 # Built With
 
 - Java
 - Java FX
 - MySql
 - Jasper Reports
 - And a little bit of CSS
